The Ripple Effect:
Benchmarking Repository-Wide
Collaborative Code Editing

A benchmark for evaluating LLMs on multi-target collaborative code editing triggered by a single initial change across real-world repositories.

GitHub Dataset & Code
Benchmark

In real-world development, a Trigger Edit often causes a chain reaction — requiring coordinated Target Edits across multiple files to preserve correctness. CoEdBench is the first benchmark explicitly modeling this repository-level collaborative editing process.

Pipeline Framework

Co-Editing Pipeline

Oracle Context provided. Evaluates the upper-bound of localization and generation under ideal retrieval conditions. Two-stage: Impact Localization → Location-Wise Generation.
Agent Framework

Co-Editing Agent

Only Requirement + Trigger Edit given. Model autonomously explores the full repository using GrepTool, ViewCode, StringReplace tools. End-to-end evaluation, ≤30 iterations.
